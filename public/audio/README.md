# Dossier Fichiers Audio / Audio Tracks Folder

Déposez vos fichiers audio ici pour qu'ils soient lus par le lecteur de la page Playground :
Place your audio files here to make them playable inside the Playground's Music Player:

- `track-01.mp3` (mappé sur "Night Lines")
- `track-02.mp3` (mappé sur "Purple Motion")
- `track-03.mp3` (mappé sur "Soft Signal")

Formats acceptés : `.mp3`, `.wav`, `.m4a`
